## Environment Variables

[//]: # (BEGIN ENV_VARS)


### `PGT_LOG_PATH`

 The directory where the Daemon logs will be saved.

### `PGT_LOG_PREFIX_NAME`

 A prefix that's added to the name of the log. Default: `server.log.`

### `PGT_CONFIG_PATH`

 A path to the configuration file
